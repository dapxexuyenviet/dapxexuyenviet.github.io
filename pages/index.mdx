# Giới thiệu

## Đạp xe xuyên Việt

Trang web này dùng để tổng hợp lại kiến thức và kinh nghiệm cho việc tổ chức hành trình Đạp xe xuyên Việt.  
Các bạn có thể đóng góp theo hai cách:

1. Để lại câu hỏi và góp ý >> nhấn vào `Question? Give us feedback →`  
   _(phía bên phải màn hình, chỉ hiển thị trên màn hình máy tính)_
2. Trực tiếp chỉnh sửa trang >> nhấn vào `Edit this page` (bạn cần biết [cách sử dụng github](https://docs.github.com/en/repositories/working-with-files/managing-files/editing-files))

Bạn cũng có thể truy cập trực tiếp source code của trang web này tại đây: https://github.com/dapxexuyenviet/dapxexuyenviet.github.io  
Mình xây dựng trang web dựa trên [nextra document theme](https://nextra.site/), deploy lên [github pages](https://pages.github.com/) thông qua [github action](https://docs.github.com/en/actions).
